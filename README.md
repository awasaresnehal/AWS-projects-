# 🌐 Static Website Hosting on AWS S3

##  Description
This project demonstrates how to host a static website using AWS S3. 
The website is publicly accessible and deployed using cloud storage.

##  Features
- Static website hosting
- Public access configuration
- Fast and reliable cloud deployment

##  Technologies Used
- AWS S3
- HTML
- CSS

## Setup Instructions

1. Create an AWS account
2. Open S3 service
3. Create a bucket
4. Upload index.html file
5. Enable static website hosting
6. Add bucket policy for public access

##  Usage
Open the website using the S3 endpoint URL provided after deployment.
